Mis Trabajos en Desafío Latam

Desafío 1 CV

En este primer desafío, había que tener la habilidad y creatividad al usar HTML para diseñar y desarrollar nuestro propio CV. Fue una oportunidad única para aprender a presentar nuestras habilidades y experiencias de una manera que destaque en el mundo digital. Este desafío no solo puso a prueba nuestro conocimiento técnico, sino que también permitió expresar nuestra identidad personal a través de código, mostrando la importancia de la personalización en la web.

Desafío 2 Landing

El segundo desafío elevó la competencia al combinar HTML5 y CSS para crear una landing page de tienda online. Tuvimos la oportunidad de pensar como diseñadores web y desarrolladores, enfrentándonos al reto de crear una página que fuera estéticamente atractiva. Este desafío sirvió como un excelente escaparate para las habilidades en la creación de sitios web responsivos y atractivos.

Desafío 3 Flexbox

El tercer desafío llevó a explorar nuevas alturas en el diseño web con el uso de HTML, CSS, y especialmente Flexbox, enfocándose en la creación de una página dedicada a las iguanas. Este desafío no solo puso a prueba la capacidad para construir sitios web responsivos y visualmente atractivos, sino que también exigió pensar en cómo estructurar el contenido de manera efectiva para captar la atención y transmitir información sobre estas fascinantes criaturas.
